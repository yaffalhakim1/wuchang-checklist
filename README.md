# Wuchang: Fallen Feathers — Journey Scroll

A browser-based, single-page area-progression checklist for the action RPG **Wuchang: Fallen Feathers**. Track every shrine, boss fight, key item, and optional encounter across all eleven parts of the game — with your progress automatically saved between sessions.

## Features

- **79 tasks** organised across **11 parts** of the main journey
- **Task tags** for quick identification:
  - 🔴 **Boss** — required boss encounters
  - 🟡 **Optional** — side bosses and optional content
  - 🟢 **Shrine** — checkpoint shrines to claim
  - 🔵 **Key Item** — important items and story gates
- **Progress bar** showing overall completion
- **Per-section completion badges** that update as you check off tasks
- **Persistent progress** stored in browser `localStorage` — no account needed
- **Reset button** to start fresh for a new playthrough
- Collapsible sections to keep the view tidy
- Thematic parchment-and-ink aesthetic inspired by the game's Chinese historical setting

## Usage

Open `index.html` in any modern web browser — no build step, no dependencies, no server required.

```bash
open index.html        # macOS
xdg-open index.html    # Linux
start index.html       # Windows
```

Click any task row to mark it complete (or incomplete). Progress is saved automatically.
